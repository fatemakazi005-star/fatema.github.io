## 🚀 How to Deploy on GitHub Pages

1. **Create a new repository** on GitHub named `portfolio`
2. **Upload all files** keeping the folder structure intact
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch → `/ (root)`
5. Click **Save** — your site will be live at:  
   `https://yourusername.github.io/portfolio`

## 🖼️ Adding Your Real Images

Replace the emoji placeholders with your actual photos:

```html
<!-- In index.html, find the hero avatar and replace: -->
🎓
<!-- with: -->
<img src="assets/images/profile.jpg" alt="Fatema Kazi" />

<!-- For gallery items, replace emoji with: -->
<img src="assets/images/gallery/cert1.jpg" alt="Certificate" />
```
